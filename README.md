# phonepe-data-analysis
PhonePe Data Analysis (Python + SQL)
🔍 Project Overview

This project performs end-to-end data analysis on PhonePe transaction data.
Python is used for data cleaning and preprocessing, while SQL Server is used to extract meaningful business insights.

🛠 Tech Stack
Python (Pandas)
SQL Server
Excel / CSV Dataset
Power BI (optional)
🔄 Workflow
Collected raw transaction data
Cleaned and preprocessed data using Python
Stored cleaned data in SQL Server
Performed SQL analysis to generate insights
🧹 Data Cleaning (Python)
Removed duplicate records
Handled missing values
Converted data types (date, numeric)
Exported cleaned dataset for SQL analysis

📂 File: notebooks/data_cleaning.ipynb

📊 SQL Analysis

Key analysis performed:

Total transaction value
State-wise performance
Transaction type analysis
Year-wise growth trends
Quarter-wise trends
District and pincode-level insights

📂 File: sql/analysis.sql

📈 Key Insights
High transaction volume indicates strong adoption of digital payments
A few states contribute the majority of transactions
Certain transaction types dominate user activity
Consistent year-wise growth shows increasing engagement
Local-level analysis highlights expansion opportunities
